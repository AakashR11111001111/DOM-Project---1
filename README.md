# DOM Project 1

This is a simple DOM manipulation project where styles are applied to a div element using JavaScript.

## Project Overview

The project dynamically styles a `div` element with the following properties:
- Background color: Yellow
- Margin: 20px
- Padding: 10px
- Font size: 18px
- Font weight: Bold
- Height: 200px
- Width: 300px
